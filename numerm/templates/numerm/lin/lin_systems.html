Systems of Linear Equations

We're familiar with linear equations: ${y = mx + b,}$ ${ax + by + c = 0,}$ ${y = cb,}$ these equations all form lines on the Cartesian plane. In the linear algebra kingdom, we move beyond the flatland of a 2-dimensional plane into a world of higher dimensions.

If we tried finding all the solutions, we might be at it for a while. The equation has an infinite number of solutions: ${x_1 = 8 - 2s + 3t,}$ where ${x_2 = s,}$ ${x_3 = t,}$ and ${s, t \in \R.}$

A set of linear equations is called a system of linear equations, or more concisely, a linear system. Here's an example of a linear system:

Linear algebra is the investigation of such systems, where the predominant questions include, but are not limited to: Does the linear system have any solutions? Does the system have no solution, a unique solution, or an infinite number of solutions? If there are solutions, how do we find them? Do the solutions have a structure or pattern? These questions will guide our exploration of the linear algebra world.

As we can see, once we venture beyond two variables, graphical representations become difficult. The key idea, however, is that linear equations of three variables represent a plane. Each row in the matrix above gives us a plane in three dimensions. The intersection of these planes embodies the solution to the linear system. From geometry, we know that the intersection of two planes is a line. The intersection of three planes, however, is either a line or a point. In our case, it's like a point.
