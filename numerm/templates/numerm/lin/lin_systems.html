Gauss's Method

Given a linear system ${L,}$ the following operations do not change ${L}$'s set of solutions:

  1. Swapping - Moving an equation to another position in the system.
  2. Scaling - Multiplying both sides of an equation in the system by a nonzero constant.
  3. Pivoting - Replacing an equation by the sum of (a) itself, and (b) the multiple of another equation in the system.

Definition: Homogeneous System

An ${m \times n}$ system is homogenous if ${b_i = 0}$ for all ${i = 1, 2, \ldots, n.}$

Notice that because a system is homogenous, we have the following lemma:

lemma

If a system is homogeneous, then ${x_1 = x_2 = \ldots = x_n = 0}$ is always a solution.

Definition: System Specificity

Given an ${m \times n}$ linear system,

  • If ${n > m,}$ then ${m \times n}$ is an underspecified system.
  • If ${n < m,}$ then ${m \times n}$ is an overspecified system.
