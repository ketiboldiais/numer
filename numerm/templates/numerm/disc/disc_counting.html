Counting

In these next few sections, we explore the basics of counting. The concept of counting might seem intuitive, but looks can be deceiving. Many counting problems require us to first determine (a) what exactly it is we are counting, and (b) how we should count. It is these problems that concern the next sections.

Before we begin counting, however, we must revisit some aspects of basic set theory. Like many other areas of mathematics, counting depends heavily on set theoretic foundations.

Basic Set Theory

A set is simply a collection of distinct objects. For example, ${\{ 1, 2, 3, 4, 5 \}}$ is a set. Contrary to what may be taught in high school courses, ${\{ 1, 1, 1, 2 \}}$ is a bag, not a set. A bag, also called a multiset or mset, is a collection of distinct and non-distinct objects; i.e., there may be more than one instance of each of its elements. We are concerned here primarily with sets.

Set Membership. The objects in a set are called that set's members. For example, the set of all whales includes humpback whales and blue whales. To denote such membership, we use the symbol ${\in.}$ In the example above, we would write ${\text{blue whales} \in \text{whales}.}$

The Empty Set.Sets are defined by membership rules. All objects that satisfy that rule are members of that set. For example, the set of all cats includes Siamese and Persians.

However, some membership rules imply that the set contains no members. For example, the set of all immortal humans. Setting aside questions of the living dead, this set contains no members, for all humans are mortal (at least as of now). This phenomenon leads to a special kind of set — the empty set, or the set containing no objects.

Equality of Sets. Suppose ${A}$ is the set ${\{ 1, 2, 3 \}}$ and ${B}$ is the set ${\{ 1, 2, 3 \}.}$ Because ${A}$ and ${B}$ have the exact same members, we say that they are equal.

Similarly, ${\{ 4 5 6 \}}$ and ${\{ 6 4 5 \}}$ are equal sets. Why? Because the definition of a set imposes no requirement of order. A set is simply a collection of distinct objects — nothing more.

Well, we know the first multiple of three between 62 and 215 is 63. The last multiple of three is 213. Thus, the actual array we want to count is ${63, 66, 69, \ldots, 213.}$ Now, we need to convert this array into one we know how to count, so we divide each element in the array by 3: ${21, 22, 23, \ldots , 71.}$ From our previous proposition, we know how to count this array: ${(71 - 21) + 1 = 51.}$ Thus, there are 51 numbers in the array — i.e., there are 51 multiples of 3 between 62 and 215.

Let's try another one. How many multiples of 10 are between 9 and 101? Again, the first multiple of 10 is 10. The last multiple of 10 is 100. Thus, the sequence is ${[10, 20, \ldots, 100].}$ We divide by 10: ${[1, 2, 3, \ldots, 10].}$ And again, we know how to count this sequence: ${(10 - 1) + 1 = 10.}$ There are 10 multiples of 10 between 9 and 101.

How many multiples of 10 are between 11 and 103? Here, the first multiple of 10 is 20. The last multiple of 10 is 100. We have the sequence ${[20, 30, \ldots, 100].}$ Dividing by 3: ${[2, 3, 4, \ldots, 10.]}$ Then counting: ${(10-2) + 1 = 9.}$ There are 9 multiples of 10 between 11 and 103.

Now that we know how to count multiples, we can start counting even more elaborate sequences: How many four-digit positive integers are perfect cubes?

Solution: We know that the first four-digit positive integer that is a perfect cube is ${10 \cdot 10 \cdot 10 = 1000.}$ The last four-digit perfect cube is a bit trickier to find. Fortunately, the method of exhaustion is feasible here. We know ${20 \cdot 20 \cdot 20 = 8000.}$ This is already pretty close to five-digits, so let's just try the next positive integer: ${21 \cdot 21 \cdot 21 = 9,261.}$ Now we're very close; 22 will probably enter five-digit territory: ${22 \cdot 22 \cdot 22 = 10,648.}$ Great, so we know our sequence is ${[1000, \ldots, 9261].}$ Taking the cubic root of each element, we have ${[10, \ldots, 21.]}$ Counting this list: ${(21 - 10) + 1 = 12.}$ Hence, there are 12 four-digit positive integers that are perfect cubes.

Exercise. How many numbers are in the following sequences:

  1. ${L = [36, 37, 38, \ldots , 92, 93]}$
    2. Solution

    We use the relevant proposition: ${(93 - 36) + 1 = 58.}$ There are 58 numbers in the sequence ${L;}$ ${n(L) = 58.}$

  2. ${N = [4, 6, 8, \ldots, 128, 130]}$
    3. Solution

    Again, we convert the sequence into one we can actually count. We can see each element is a multiple of 2. Thus, the sequence is: ${[(4/2), (6/2), (8/2), \ldots, (128/2), (130/2)].}$ Simplifying: ${2, 3, 4, \ldots, 64, 65.}$ Using the proposition: ${(65 - 2) + 1 = 64.}$ Thus, ${n(N) = 64.}$.

  3. ${S = [-33, -28, -23, \ldots, 52, 57]}$
    4. Solution

    Again, we convert the sequence into one we can count. We can see that each element in the sequence differs by 5. To simplify the sequence so we can divide by 5, let's add 3 to each element: ${[-30, -25, -20, \ldots, 55, 60.]}$ Now let's divide by 5: ${-6, -5, -4, \ldots, 11, 12.}$ Using our formula: ${(12 - (-6)) + 1 = 19.}$ Thus, ${n(S) = 19.}$

  4. ${Q = [147, 144, 141, \ldots, 42, 39]}$
    5. Solution

    In this sequence, we can see each element differs by 3. Let's try subtracting 9 from each element: ${[30, 33, \ldots , 132, 135, 138.]}$ Reverse the list and divde by 3: ${[10, 11, \ldots, 44, 45, 46.]}$ Using our proposition: ${(46-10) + 1 = 37.}$ Thus, ${n(Q) = 37.}$

  5. ${C = [3 \frac{2}{3}, 4 \frac{1}{3}, 5, 5 \frac{2}{3}, \ldots , 26 \frac{1}{3}, 27]}$
    6. Solution

    We can see each element differs by ${\frac{2}{3}.}$ Let's try and give all of the elements a common denominator: ${[\frac{11}{3}, \frac{13}{3}, \frac{15}{3}, \ldots, \frac{79}{3}, \frac{81}{3}].}$ Since each of the elements has a multiplier of ${\frac{1}{3},}$ we can factor ${\frac{1}{3}}$ out, leaving us with: ${[11, 13, 15, \ldots, 79, 81.]}$ These are all separated by 2, so let's subtract 1 to each element: ${[10, 12, 14, \ldots, 78, 80];}$ then divide by 2: ${[5, 6, 7, \ldots, 39, 40.]}$ We can count this list: ${(40-5) + 1 = 36.}$ Thus, ${n(C) = 36.}$

Exercise. How many odd perfect squares are between 5 and 211?

Solution

Again, let's come up with a sequence. First, let's find the first element. The first element is 9: ${3^2 = 9.}$ Now let's find the last element. We know ${12^2 = 144.}$ So just a little higher. ${13^2 = 169.}$ ${15 \times 15 = 225;}$ too high. ${14^2 = 196,}$ but this is even. So, the first element is 9, and the last element is 169. Thus, we have ${[9, 25, 49, 81, \ldots 196].}$ Taking the square root each element: ${[3, 5, 7, 9, \ldots , 13.]}$ Subtracting 1 from each element: ${[2, 4, 6, 8, \ldots , 12.]}$ Dividing each element by 2: ${[1, 2, 3, 4, \ldots, 6.]}$ Thus, there are 6 odd perfect squares between 5 and 211.

Counting with Addition and Subtraction

Counting is deeply intertwined with arithmetic. Consider the following problem:

12 students are taking at least 1 foreign language. The school offers only Spanish and French. 8 students take Spanish, 5 take both Spanish and French. How many students take French?

There are multiple ways to approach this problem, but the worst possible way is to blindly add and subtract. We want to think as logically as possible. More importantly, we want to think about this problem in terms of set theory. Here, we have a situation where students take Spanish, French, or both Spanish and French. Now, we know ${n(S) = 14,}$ and we know that ${n(S) = n(K \cap S) + n(A \cap S).}$ Hence, ${n(K \cap S) = n(S) - n(A \cap S) = 14 - 11 = 3.}$ Therefore, there are 3 short-hair kittens.
