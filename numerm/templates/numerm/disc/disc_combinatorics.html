Enumeration

In these next few sections, we explore the basics of enumeration, the formal term for what we think of when we hear the word “counting.” The concept of counting things might seem intuitive, but looks can be deceiving. Many counting problems require us to first determine (a) what exactly it is we are counting, and (b) how we should count. It is these problems that concern the next sections.

We can use the proposition above for even more complex problems. For example, how many multiples of 3 are between 62 and 215?

Well, we know the first multiple of three between 62 and 215 is 63. The last multiple of three is 213. Thus, the actual array we want to count is ${63, 66, 69, \ldots, 213.}$ Now, we need to convert this array into one we know how to count, so we divide each element in the array by 3: ${21, 22, 23, \ldots , 71.}$ From our previous proposition, we know how to count this array: ${(71 - 21) + 1 = 51.}$ Thus, there are 51 numbers in the array — i.e., there are 51 multiples of 3 between 62 and 215.

Let's try another one. How many multiples of 10 are between 9 and 101? Again, the first multiple of 10 is 10. The last multiple of 10 is 100. Thus, the sequence is ${[10, 20, \ldots, 100].}$ We divide by 10: ${[1, 2, 3, \ldots, 10].}$ And again, we know how to count this sequence: ${(10 - 1) + 1 = 10.}$ There are 10 multiples of 10 between 9 and 101.

How many multiples of 10 are between 11 and 103? Here, the first multiple of 10 is 20. The last multiple of 10 is 100. We have the sequence ${[20, 30, \ldots, 100].}$ Dividing by 3: ${[2, 3, 4, \ldots, 10.]}$ Then counting: ${(10-2) + 1 = 9.}$ There are 9 multiples of 10 between 11 and 103.

Now that we know how to count multiples, we can start counting even more elaborate sequences: How many four-digit positive integers are perfect cubes?

Solution: We know that the first four-digit positive integer that is a perfect cube is ${10 \cdot 10 \cdot 10 = 1000.}$ The last four-digit perfect cube is a bit trickier to find. Fortunately, the method of exhaustion is feasible here. We know ${20 \cdot 20 \cdot 20 = 8000.}$ This is already pretty close to five-digits, so let's just try the next positive integer: ${21 \cdot 21 \cdot 21 = 9,261.}$ Now we're very close; 22 will probably enter five-digit territory: ${22 \cdot 22 \cdot 22 = 10,648.}$ Great, so we know our sequence is ${[1000, \ldots, 9261].}$ Taking the cubic root of each element, we have ${[10, \ldots, 21.]}$ Counting this list: ${(21 - 10) + 1 = 12.}$ Hence, there are 12 four-digit positive integers that are perfect cubes.

Let's consider another interesting example. There are four villages, ${A,}$ ${B,}$ ${C,}$ and ${D.}$ Each line represents a path between the villages. How many ways are there to go from ${A}$ to ${D}$ along the paths, if one can only move left to right? (I.e., one cannot move ${ABABABACD;}$ only ${ACD}$ or ${ABD}$).

This is another counting problem. First, observe the number of paths between each village. Between ${A}$ and ${C,}$ there are 2 paths, and between ${A}$ and ${C,}$ there are 5 paths. Accordingly, there are 10 possible paths if we take the route ${ACD.}$ Now consider the route ${ABD.}$ From ${A}$ to ${B,}$ there are 4 possible paths, and from ${B}$ to ${D,}$ there are 3 possible paths. Hence, there are 12 possible paths for taking the route ${ABD.}$ In total, there are ${10 + 12 = 22}$ possible paths from ${A}$ to ${D.}$

  1. How many numbers are in the following sequences:
    1. ${L = [36, 37, 38, \ldots , 92, 93]}$
      2. Solution

      We use the relevant proposition: ${(93 - 36) + 1 = 58.}$ There are 58 numbers in the sequence ${L;}$ ${n(L) = 58.}$

    2. ${N = [4, 6, 8, \ldots, 128, 130]}$
      3. Solution

      Again, we convert the sequence into one we can actually count. We can see each element is a multiple of 2. Thus, the sequence is: ${[(4/2), (6/2), (8/2), \ldots, (128/2), (130/2)].}$ Simplifying: ${2, 3, 4, \ldots, 64, 65.}$ Using the proposition: ${(65 - 2) + 1 = 64.}$ Thus, ${n(N) = 64.}$.

    3. ${S = [-33, -28, -23, \ldots, 52, 57]}$
      4. Solution

      Again, we convert the sequence into one we can count. We can see that each element in the sequence differs by 5. To simplify the sequence so we can divide by 5, let's add 3 to each element: ${[-30, -25, -20, \ldots, 55, 60.]}$ Now let's divide by 5: ${-6, -5, -4, \ldots, 11, 12.}$ Using our formula: ${(12 - (-6)) + 1 = 19.}$ Thus, ${n(S) = 19.}$

    4. ${Q = [147, 144, 141, \ldots, 42, 39]}$
      5. Solution

      In this sequence, we can see each element differs by 3. Let's try subtracting 9 from each element: ${[30, 33, \ldots , 132, 135, 138.]}$ Reverse the list and divde by 3: ${[10, 11, \ldots, 44, 45, 46.]}$ Using our proposition: ${(46-10) + 1 = 37.}$ Thus, ${n(Q) = 37.}$

    5. ${C = [3 \frac{2}{3}, 4 \frac{1}{3}, 5, 5 \frac{2}{3}, \ldots , 26 \frac{1}{3}, 27]}$
      6. Solution

      We can see each element differs by ${\frac{2}{3}.}$ Let's try and give all of the elements a common denominator: ${[\frac{11}{3}, \frac{13}{3}, \frac{15}{3}, \ldots, \frac{79}{3}, \frac{81}{3}].}$ Since each of the elements has a multiplier of ${\frac{1}{3},}$ we can factor ${\frac{1}{3}}$ out, leaving us with: ${[11, 13, 15, \ldots, 79, 81.]}$ These are all separated by 2, so let's subtract 1 to each element: ${[10, 12, 14, \ldots, 78, 80];}$ then divide by 2: ${[5, 6, 7, \ldots, 39, 40.]}$ We can count this list: ${(40-5) + 1 = 36.}$ Thus, ${n(C) = 36.}$

  3. Suppose a graph consists of ${n}$ points. How many different functions are defined on these ${n}$ points if each function is either ${0,}$ ${1,}$ or ${\texttt{null}?}$
    Solution

    When dealing with problems involving abstractions, it's often best to consider a simpler, concrete case first. Suppose the function is called ${f.}$ Next, instead of considering ${n}$ points, suppose there is just 1 point. With just 1 point, we have ${f(1) = -1,}$ ${f(1) = 0,}$ ${f(1) = \texttt{null}.}$ This means that with just 1 point, we have 3 possible functions. Because we're dealing with a function, the value after ${f(1)}$ — the value of ${f(2)}$ — could be either ${-1,}$ ${0,}$ or ${\texttt{null}.}$ This means that we have ${3 \cdot 3 = 3^2 = 9}$ possible functions. With 3 points, we have ${3 \cdot 3 \cdot 3 = 3^3 = 27.}$ Expanding on this thought process, there are ${3^n}$ possible functions.

  4. Blackacre Corp. has two divisions, software and hardware. Blackacre owns a building with 12 floors. Assuming Blackacre will assign 1 floor to each division, how many different ways are there to assign different floors to each division?
