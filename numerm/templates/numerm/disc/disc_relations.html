Relations

As humans, we perceive the world according to the way things relate to one another. Corporate hierarchy, family relationships, friend networks, court systems, if-then, if-and-only-if, cause and effect, these are all relationships. Mathematical objects are no different — they too have relationships to other mathematical objects. The set $A$ is related to the set $B$ if ${ A \subseteq B }$. The number $a$ is related to the number $b$ if ${ a < b }$, or if ${ a^2 + b^2 = 1 }$.

In mathematics, a relation is a proposition that states how a set of objects relates to another set of objects. For example, suppose we have two sets: ${A = \{ 0, 1, 2 \}}$ and ${B = \{ 1, 2, 3 \}}$. Suppose further that an element ${a \in A}$ is related to an element ${b \in B}$, if, and only if, ${a < b}$. From this rule, we can infer:

  1. ${0 \mathrel{R} 1}$ since ${0 < 1,}$
  2. ${0 \mathrel{R} 2}$, since ${0 < 3,}$
  3. ${1 \mathrel{R} 2}$, since ${1 < 2,}$
  4. ${1 \mathrel{R} 3}$, since ${1 < 3,}$
  5. ${2 \mathrel{R} 3}$ since ${2 < 3,}$
  6. ${1 \not\mathrel{R} 1}$, since ${1 \nless 1,}$
  7. ${2 \not\mathrel{R} 1}$, since ${2 \nless 1,}$ and
  8. ${2 \not\mathrel{R} 2}$, since ${2 \nless 2}$

We use the notation ${a \mathrel{R} b}$ to express the proposition ${(a,b) \in \space \mathrel{R}}$ and ${a \not\mathrel{R} b}$ to express the proposition ${(a, b) \notin \space \mathrel{R}.}$ If ${(a,b) \in \space \mathrel{R},}$ we say that ${a}$ is related to ${b}$ by ${\mathrel{R}.}$

      • ${(2,2) \implies \dfrac{2-2}{2} = \dfrac{0}{2} = 0}$. ${0 \in \mathbb{Z}}$, so ${(2,2) \in \space \mathrel{R}}$.
      • ${(2,3) \implies \dfrac{2-3}{2} = \dfrac{-1}{2}}$. ${- \frac{1}{2} \notin \mathbb{Z}}$, so ${(2,3) \notin \space \mathrel{R}}$.

Relations on a Set

Relations from a set ${S}$ to itself are called relations on a set. In the previous examples, we considered binary relations, which always involve two sets, say ${A}$ and ${B.}$ With a relation on a set, we're concerned with just a single a set, and the relationships between the set's members. Suppose we had a set ${S = \{ 1, 2, 3, 4 \}.}$ Now suppose the following relationships:

  1. ${\mathrel{R}_1 = (1, 2)}$
  2. ${\mathrel{R}_2 = (2, 1)}$
  3. ${\mathrel{R}_3 = (3, 1)}$
  4. ${\mathrel{R}_4 = (3, 3)}$
  5. ${\mathrel{R}_5 = (3, 4)}$
  6. ${\mathrel{R}_6 = (4, 4)}$

All of these relationships, put together, form a relation on ${A.}$ Here, we see that ${1}$ has a relationship with ${2,}$ and ${2}$ has a relationship with ${1.}$ ${3}$ has a relationship with ${1,}$ but ${1}$ does not have relationship with ${3.}$ ${3}$ has a relationship with ${4,}$ but ${4}$ does not have a relationship with ${3.}$ Both ${3}$ and ${4}$ have relationships with themselves. This type of relationship is called a loop.

Properties of Relations

In the previous examples, we saw how some relations have direction, whether that's to other objects or to themselves. This is evidence of the fact that relations have properties. We examine each of these properties below.

Reflexive Relations

Let's say we gathered all of the people in the world into a set ${A,}$ and constructed another set containing the same members, call it ${B.}$ Now suppose we construct the Cartesian product ${A \times B.}$ The set ${A \times B}$ consists of the members ${(a, b),}$ where ${a \in A}$ and ${b \in B.}$ Finally, we define a relation: ${R}$ consisting of ${(a, b)}$ where ${a}$ and ${b}$ have the same mother and the same father. It follows that ${a \mathrel{R} a}$ for every person ${a.}$ This is the property of reflexivity.

For the relation to be irreflexive, there must be no loops entirely. For example, the relation ${<}$ is irreflexive on the integers, since no integer is less than itself. Otherwise, the relation is either reflexive or not reflexive.

Symmetric Relations

Suppose ${x}$ is the sibling of ${y.}$ Is ${y}$ the sibling of ${x?}$ Yes. This is an example of a symmetric relation: A relation where some element ${a}$ is related to a second element ${b}$ if and only if the second element ${b}$ is also related to the first element ${a.}$ Explicitly:

Antisymmetric Relations

Now suppose ${x}$ is the father of ${y.}$ Is ${y}$ the father of ${x?}$ Of course not. The “father of” relation is an example of an antisymmetric relation: A relation where there are no pairs of ${x}$ and ${y}$ where ${x}$ is related to ${y}$ and ${y}$ is related to ${x.}$

The Circle Relation. One example of a relation is the circle relation. We can define it as follows: Let $C$ be a relation from ${\mathbb{R}}$ to ${\mathbb{R}.}$ For any ${(x,y) \in \mathbb{R} \times \mathbb{R},}$ ${(x,y) \in C}$ means that ${x^2 + y^2 = 1.}$

Here, the domain of $C$ is all real numbers, ${\mathbb{R}}$. Its codmain is also ${\mathbb{R}.}$
