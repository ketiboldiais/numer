Relations

As humans, we perceive the world according to the way things relate to one another. Corporate hierarchy, family relationships, friend networks, court systems, if-then, if-and-only-if, cause and effect, these are all relationships. Mathematical objects are no different — they too have relationships to other mathematical objects. The set $A$ is related to the set $B$ if ${ A \subseteq B }$. The number $a$ is related to the number $b$ if ${ a < b }$, or if ${ a^2 + b^2 = 1 }$.

If ${(x, y) \in \mathrel{R},}$ we say that ${x}$ is related to ${y}$ by ${\mathrel{R}.}$ Importantly, notice the relation ${\mathrel{R}}$ concerns only two sets, ${A}$ and ${B.}$ Hence our defining the relation as a binary relation, rather than just simply “relation.” We include the distinction “binary” because there are many kinds of relations, and a relation between two sets — a binary relation — is just one of them.

    What ordered pairs in ${A \times B}$ are in $\mathrel{R}$?

    Solution
    • ${A \times B = \{ (1,1), (1,2), (1,3), (2,1), (2,2), (2,3) \}}$.
    • To determine which of these ordered pairs are in $\mathrel{R}$, we must test each ordered pair for whether the relation's definition is satisfied.
      • ${(1,1) \implies \dfrac{1-1}{2} = \dfrac{0}{2} = 0}$. ${0 \in \mathbb{Z}}$, so ${(1,1) \in \mathrel{R}}$.
      • ${(1,2) \implies \dfrac{1-2}{2} = \dfrac{-1}{2} = -\dfrac{1}{2}}$. $-{\frac{1}{2} \notin \mathbb{Z}}$, so ${(1,2) \notin \mathrel{R}}$.
      • ${(1,3) \implies \dfrac{1-3}{2} = \dfrac{-2}{2} = -1}$. ${-1 \in \mathbb{Z}}$, so ${(1,3) \in \mathrel{R}}$.
      • ${(2,1) \implies \dfrac{2-1}{2} = \dfrac{1}{2}}$. ${\frac{1}{2} \notin \mathbb{Z}}$, so ${(2,1) \notin \mathrel{R}}$.
      • ${(2,2) \implies \dfrac{2-2}{2} = \dfrac{0}{2} = 0}$. ${0 \in \mathbb{Z}}$, so ${(2,2) \in \mathrel{R}}$.
      • ${(2,3) \implies \dfrac{2-3}{2} = \dfrac{-1}{2}}$. ${- \frac{1}{2} \notin \mathbb{Z}}$, so ${(2,3) \notin \mathrel{R}}$.
    • ∴ ${\mathrel{R} = \{ (1,1), (1,3), (2,2) \}}$. Or, more explicitly: ${1 \mathrel{R} 1}$, ${1 \mathrel{R} 3}$, ${2 \mathrel{R} 2}$.

    ${\mathrel{R} : \{ 1,2 \} \mapsto \{ 1,2,3 \}}$

Relations on a Set

Relations from a set ${S}$ to itself are called relations on a set. In the previous examples, we considered binary relations, which always involve two sets, say ${A}$ and ${B.}$ With a relation on a set, we're concerned with just a single a set, and the relationships between the set's members. Suppose we had a set ${S = \{ 1, 2, 3, 4 \}.}$ Now suppose the following relationships:

  1. ${\mathrel{R}_1 = (1, 2)}$
  2. ${\mathrel{R}_2 = (2, 1)}$
  3. ${\mathrel{R}_3 = (3, 1)}$
  4. ${\mathrel{R}_4 = (3, 3)}$
  5. ${\mathrel{R}_5 = (3, 4)}$
  6. ${\mathrel{R}_6 = (4, 4)}$

This type of relationship is called a loop.

We can take some arbitrarily complex structure — an outline for a class, the characters in a novel, the novel itself, a legal statute, a classification system — and express it as a relation.

The Circle Relation. One example of a relation is the circle relation. We can define it as follows: Let $C$ be a relation from ${\mathbb{R}}$ to ${\mathbb{R}.}$ For any ${(x,y) \in \mathbb{R} \times \mathbb{R},}$ ${(x,y) \in C}$ means that ${x^2 + y^2 = 1.}$

Here, the domain of $C$ is all real numbers, ${\mathbb{R}}$. Its codmain is also ${\mathbb{R}.}$
