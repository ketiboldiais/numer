The property above is immensely useful when we counter an open sentence of the form ${a + x = a.}$ For this sentence to be a true proposition, it must be the case that ${x = 0.}$ Why? Because: If ${a + x = a,}$ then by subtracting ${a}$ from both sides, we have ${(-a) + (a + x) = (-a) + a = 0.}$ Thus, ${((-a) + a) + x = 0.}$ Simplifying further, ${0 + x = 0.}$ Ergo, ${x = 0.}$

We might ask, why write ${+ (-a)}$? Why not just write ${-a?}$ Fair questions, and we very well could have just written ${-a.}$ However, we write ${(-a)}$ to demonstrate the idea that subtraction is really just convenient construct for adding negative numbers. In other words, subtraction is, at its core, addition. The expression ${a - b}$ is logically equivalent fo ${a + (-b).}$
