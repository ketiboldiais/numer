Algebra Appendix

Set Theory

Definition: Set. A set is any well-defined collection of objects. By well-defined, we mean that given any object in the world, we can determine either (a) the object belongs to the set or (b) the object does not belong to the set.

Definition: Element. Let ${S}$ be any set. The objects belonging to ${S}$ are called the elements, or members, of ${S.}$ If ${x}$ is a member of ${S,}$ we write ${x \in S.}$ If ${x}$ does not belong to ${S,}$ we write ${x \notin S.}$

Definition: The Empty Set. The set containing no elements is called the empty set. It is denoted with the symbol ${\varnothing.}$

Definition: Equal Sets. Two sets ${A}$ and ${B}$ are equal if, and only if:

If ${A}$ and ${B}$ are equal, we write ${A = B.}$ If ${A}$ and ${B}$ are not equal, we write ${A \neq B.}$

Definition: Finite Set. A set with a definite number of elements is called a finite set.

Definition: Infinite Set. A set without a definite number of elements is called a infinite set.

Definition: Class. A set whose members are sets is called a class.

Definition: Indexed Class. A class ${\mathscr{C}}$ is called an indexed class if there exists a set ${S}$ such that for each element ${i \in S,}$ there exists a unique member ${A_i}$ in ${\mathscr{C}}$ associated with ${i}$ and ${\mathscr{C} = \{ A_i : i \in S \}.}$ In this case, the set ${S}$ is called the index set.

Algebra

Exponent Laws. Let ${a \in \R^+.}$ Then the following propositions are true:

  1. ${a^x a^y = a^{x + y}}$
  2. ${a^x > 0}$
  3. ${\dfrac{a^x}{a^y} = a^{x - y}}$
  4. ${\left( a^x \right)^y = a^{x \cdot y}}$
  5. ${a^{-x} = \dfrac{1}{a^x}}$
  6. ${a^0 = 1}$
  7. ${a^1 = a}$
  8. ${1^x = 1}$

Increasing Exponential Function. Let ${a \in \R.}$ If ${a > 1,}$ then ${f(x) = a^x}$ is an increasing function. That is, ${x \leq y \implies a^x \leq a^y}$

Decreasing Exponential Function. Let ${a \in \R.}$ If ${0 < a < 1,}$ then ${f(x) = a^x}$ is a decreasing function. That is, ${x \leq y \implies a^x \geq a^y}$

Exponential Injection. Let ${a \in \R.}$ If ${a > 0}$ and ${a \neq 1,}$ then ${a^x}$ is an injection. That is, ${a^x \neq a^y}$ for all ${x \neq y.}$

Theorem. Let ${a \in \R.}$ For any ${a > 0,}$ and ${a \neq 1,}$ ${a^x = 1}$ if and only if ${x = 0.}$
