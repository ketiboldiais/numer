Basic Algebra

Here, we review basic algebra skills necessary for higher level mathematics. The skills presented here are akin to elementary-level reading comprehension in mathematics, so it is absolutely critical that we have them down pat before moving any further. Before we proceed, some terminology.

For addition, the following terms are available for use:

  • ${\text{term} + \text{term} = \text{sum}}$
  • ${\text{summand} + \text{summand} = \text{sum}}$
  • ${\text{addend} + \text{addend} = \text{sum}}$
  • ${\text{augend} + \text{addend} = \text{sum}}$

For subtraction:

  • ${\text{term} - \text{term} = \text{difference}}$
  • ${\text{minuend} - \text{subtrahend} = \text{difference}}$

For multiplication:

  • ${\text{factor} \times \text{factor} = \text{product}}$
  • ${\text{multiplier}(\text{multiplicand}) = \text{product}}$

For division:

  • ${\text{dividend } \div \text{ divisor} = \text{fraction, quotient, or ratio}}$
  • ${\text{numerator } / \text{ denominator} = \text{fraction, quotient, or ratio}}$
