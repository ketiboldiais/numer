Regex

Regular expressions are some of the most useful tools in development. For example, suppose we had the HTML tag <div class="light"></div> written throughout a page. We want to replace every instance of this tag with <figure class="dark"></figure>, while keeping the contents inside the tag. We can't exactly search-and-replace with just natural language because we want to keep variable content. It seems like manually changing these tags, one by one, is the only way out. But the task is daunting — there could very well be hundreds of them. This problem is precisely what regular expressions are for. With regular expressions, we can search for the tag and replace everything other than the variable content.
