Functions

In C++, we must provide a function prototype before we actually implement the function. This prototype serves as sort of a "heads up" to the compiler about an upcoming function. It takes the form:

${t_r}$ represents the function's return type (the type of the function's output). ${f}$ is the function's name. ${t_0, \ldots, t_n}$ are the types of the function's parameters. ${p_0, \ldots, p_n}$ are the parameter.
