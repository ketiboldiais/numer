Control Flow

Like most other languages, C++ provides the fundamental control flow constructions of conditional control and loops. We'll first examine conditional control.

Conditional Control

We can achieve conditional control with if, else if, and else statements.

Loops

The for loop in C++ looks similar to most other languages: 


		for (int i = 0; i < 10; i++) {
			if (i % 2 == 2) {
				x += i;
			}
		}

So does the while loop:


		while (x > 0 && c == 'Q' || b) {
			x = x / 2;
			if (x == 42) { return 0; }
		}
