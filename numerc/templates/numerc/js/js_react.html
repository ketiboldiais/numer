React

Here is a sample React program:


		import "./styles.css";
		import React, { useState } from "react";
		import Field from './components/field';
		import Languages from './components/languages';
		import Translate from './components/translate';

		export default function App() {
			const [language, setLanguage] = useState('ru');
			const [text, setText] = useState(''); 
			return (
				<div>
					<Field label = "Enter English" onChange = {setText} value = {text}/>
					<Languages language = {language} onLanguageChange = {setLanguage}/>
					<hr />
					<Translate text = {text} language = {language}/>
				</div>
			);
		}

The first thing we focus on is the call to a function called App(). This is the application function, or app function, and it is an instance of a react component. A react component is a plain JavaScript function that accomplishes one of the following tasks:

  1. return JSX (JavaScript Syntax), or
  2. Handle user events (e.g., entering text or clicking a button)

JSX is just a set of instructions commanding informing React what content we want displayed on the screen. In the code above, the JSX consists of the lines that look like HTML code. JSX is similar to HTML in many ways. It consits of tags that are either self-closing or require a closing tag.

JSX elements can be divided into two categories, depending on the tasks they accomplish:

  1. Elements that command React to create a normal HTML element.
  2. Elements that tell React to show another component.
